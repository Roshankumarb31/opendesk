# Security Policy

If you discover a security vulnerability in OpenDesk:

- Do not open a public issue initially
- Contact the maintainer privately:
  - Email: add-your-email-here
- Provide details:
  - Description and potential impact
  - Steps to reproduce (if possible)
  - Affected version/build info

We will acknowledge your report, investigate, and work on a fix or mitigation. Responsible disclosure is appreciated.
